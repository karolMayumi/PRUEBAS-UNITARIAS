# Ejercicios de la unidad didactica Prueba Unitaria

para que funciones usar el siguiente comando

composer install 

Despues de instalar Composer crearemos la carpeta de nuestro proyecto co el siguiente codigo:

```bash
cd D: PRUEBAS UNITARIAS\
```

Ahora dentro de esa carpeta crearemos por semanas las pruebas que ejecutaremos , empezaremos con la semana07 y lo haremos con el siguiente codigo:
```bash
mkdir semana07
```
Despues de eso abrimos la carpeta principal (PRUEBAS UNITARIAS) con Visual Studio Code donde dentro de ella esta la carpeta de semana07 para ejecutar la pruebas lea el README que se encuentra en ella.
